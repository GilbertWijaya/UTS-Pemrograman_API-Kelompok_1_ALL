1. untuk melakukan generate data 
php artisan tinker
User::factory()->count(10)->create() // untuk generate 10 data
User::create(['name'=>'gilbert','email'=>'gilbert@gmail.com','password'=>'gilbert']); // untuk generate 1 data
exit // untuk keluar dari tinker

2. instalasi sactum
composer require laravel/sanctum
php artisan install:api // untuk generate file route api
php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider" // publish vendor
sisanya dapat dilihat pada file yang dikonfigurasi berupa 
config: sanctum.php, app.php
bootstrap : app.php
models : User.php
controller/http: AuthController
api.php
